# 📄 CV Application

![React](https://img.shields.io/badge/React-19-blue?logo=react)
![Vite](https://img.shields.io/badge/Vite-fast-purple?logo=vite)
![GitHub Pages](https://img.shields.io/badge/Deployed-GitHub%20Pages-green?logo=github)

A **React CV Builder** that allows users to enter personal information, education, and work experience and instantly see a **live CV preview**.

🌍 **Live Demo**
👉 https://boyanaaaa.github.io/Create-CV-Application/

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/boyanaaaa/Create-CV-Application.git
```

Go to the project folder:

```bash
cd Create-CV-Application
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
